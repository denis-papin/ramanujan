# AGENTS.md

## Oracle MCP obligatoire

Pour toute action qui concerne Oracle dans ce dépôt :

- Utiliser exclusivement les outils MCP Oracle/SQLcl disponibles.

## Attentes de travail

- Considérer `DENIS` comme la connexion Oracle par défaut pour ce projet.
- Pour les exports de schéma ou de données, privilégier des scripts SQL réexécutables et lisibles.
- Exclure les objets techniques non métiers quand ils ne sont pas nécessaires, par exemple les objets `DBTOOLS$%`.
