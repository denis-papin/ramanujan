# Symptôme
L'appel à `increment_goals` avec un identifiant de `Player` inconnu renvoyait une erreur Oracle technique au lieu d'une erreur métier.

# Attendu
The flow must return:
- `ORA-20003`
- `Player not found for id <player_id>`

# Observé
Le flux renvoyait auparavant une erreur d'exécution technique parce qu'il continuait après une mise à jour n'affectant aucune ligne.

# Cause racine
La procédure s'appuyait sur un comportement implicite d'exception au lieu de vérifier explicitement `SQL%ROWCOUNT = 0` après l'`UPDATE`.

# Stratégie de correction
Lever immédiatement l'erreur métier quand aucune ligne `player` n'est mise à jour, avant toute logique de réconciliation des `Award`.
